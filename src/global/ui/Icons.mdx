## Files

All the icons components should live inside `/src/global/ui/Icons`

There, you can create a file with the component name, like Save.tsx or Delete.tsx.

### Exporting and Importing

- In case you are creating an icon:

1. Create the Icon with a export. Like:

```tsx
export const IconName = () => {};
```

2. Add a export for the svg component in `src/global/ui/Icons/index.ts` like:

```js
export * from "./Save";
```

- In case you just want to import an Icon to your component:

1. Import the Icon where you want to use like:

```js
import { Save } from "@global/ui/Icons";
```

2. Add the svg component tag where you want it to be. Like:

```tsx
<div>
  <Save />
</div>
```

- In case you are importing a new Icon in this page:

1. import the icon like

   :

```js
import { Save } from "@global/ui/Icons";
```

2. Add another IconItem inside the already existing IconGallery tag like:

```tsx
<IconItem name="Save Changes">
  <Save name="save" />
</IconItem>
```
